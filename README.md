# Simple React Todo App

A simple Todo app that allows you to add new Todos on the board and remove Todos by clicking the them.

![preview_todo_app.png](/my-todo-app/src/preview.png?raw=true "Optional Title")

## Installation

```
git clone https://github.com/guanchen18/todo-app.git
cd my-todo-app
npm install
npm start
```
